## AI 工程能力概述

### 什么是 AI 工程能力

AI 工程能力是将 AI 技术应用于实际问题的综合能力，包括：

- **模型选择**：根据需求选型
- **系统设计**：构建可靠的 AI 系统
- **工程实现**：高效开发和部署
- **持续优化**：迭代改进性能

```python

CapabilityDimensions = {
    "技术能力": "编码、算法、系统设计",

}
```
