## 📝 文案策划标准

### 视频结构（5分钟标准）

| 段落 | 时长 | 内容 | 目的 |
|------|------|------|------|
| **开头** | 15秒 | 痛点/悬念/数据 | 抓住注意力 |
| **核心点1** | 30秒 | 是什么 | 建立认知 |
| **核心点2** | 45秒 | 为什么重要 | 深度理解 |
| **核心点3** | 60秒 | 案例/数据 | 增强信任 |
| **核心点4** | 45秒 | 怎么用 | 实用价值 |
| **行动点** | 30秒 | 怎么做 | 促使行动 |
| **结尾** | 15秒 | 金句+CTA | 转化互动 |

### 每段文案要求

```markdown
1. 开头要"炸" - 用数据/痛点/悬念
2. 中间要"实" - 用案例/数据支撑
3. 结尾要"促" - 明确的行动指令
```

### 3. 帧数同步算法 (核心!)

```javascript
// 关键：按实际语音时长分配帧数
function calculateFrames(segmentDurations, fps = 2) {
  const totalDuration = segmentDurations.reduce((a, b) => a + b, 0);
  const framePerSeg = segmentDurations.map(d => Math.ceil(d * fps));
  // 每段的帧数 = 该段实际时长 × fps
  return { totalFrames, framePerSeg, totalDuration };
}
```

## 🎯 质量标准

| 指标 | 要求 | 检测 |
|------|------|------|
| **时长** | 3-5分钟 | ffprobe |
| **同步** | <0.5秒差值 | 音频vs视频 |
| **画面** | 无黑屏/花屏 | 帧检查 |
| **音频** | 清晰无爆音 | 波形检查 |
| **文件** | >3MB | 大小检查 |
