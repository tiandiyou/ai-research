# AI视频生产质量框架 v2

> 基于实践反馈重新设计

## 一、质量标准

### 1.1 爆款视频核心要素

| 维度 | 权重 | 最低要求 |
|------|------|----------|
| 视觉吸引力 | 30% | 动态背景、配色、动画 |
| 内容价值 | 25% | 3个实用干货点 |
| 情感共鸣 | 20% | 痛点/悬念/金句 |
| 行动引导 | 15% | 明确点赞/关注 |
| 节奏把控 | 10% | 黄金3秒开头 |

### 1.2 视频结构（20秒）

```
0-3秒  │  痛点/悬念开场  │  🔥 抓住注意力
3-10秒 │  核心干货      │  💡 价值输出
10-17秒│  金句/总结     │  ✨ 强化记忆
17-20秒│  CTA引导      │  👋 互动引导
```

### 1.3 视觉规范

```yaml
视觉要求:
  背景: 动态渐变 或 粒子效果
  配色: 科技蓝(#00D4FF) + 深色底(#0A0A1A)
  文字: 主标题80pt + 副标题40pt + 强调色
  动画: 淡入淡出 + 缩放 + 脉冲
  转场: 每3秒切换场景
```

### 1.4 禁止出现

- 纯静态文字无变化
- 单调配色
- 无节奏平铺直叙
- 无情感引导

---

## 二、生产流程

```
PRD → 视觉设计 → 动画帧生成 → 合成视频 → 上传
```

---

## 三、质量检查清单

```
□ 开头3秒有吸引力？
□ 有动态视觉元素？
□ 内容有价值增量？
□ 有金句或记忆点？
□ CTA明确？
□ 总时长20秒左右？
```
