## 四、使用方法

### 4.1 环境准备

```bash
# 安装依赖
pip3 install edge-tts --break-system-packages
npm install puppeteer

# Chrome已安装
/usr/bin/google-chrome
```

### 4.2 生成视频

```bash
cd docs/video-production

# 1. 生成语音
python3 -c "
import edge_tts, asyncio
videos = [...]  # 文案配置
async def gen():
    for v in videos:
        c = edge_tts.Communicate('。'.join(v['texts']), 'zh-CN-XiaoxiaoNeural')
        await c.save(f'out/audio-{v[\"id\"]}.mp3')
asyncio.run(gen())
"

# 2. 生成视频
node gen_video.cjs

# 3. 上传Gitee
python3 upload.py
```

### 4.3 核心脚本

| 脚本 | 功能 |
|------|------|
| `gen_video.cjs` | Puppeteer生成视频帧+ffmpeg合成 |
| `render_with_audio.py` | 完整流水线（语音+视频） |
| `upload.py` | 批量上传到Gitee |
