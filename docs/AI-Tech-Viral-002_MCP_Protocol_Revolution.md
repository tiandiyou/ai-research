## 四、MCP核心概念深度解析

### 4.1 三大组件

```yaml
MCP架构：
  client: # AI应用（Claude、GPT等）
    - 发起请求
    - 管理会话
    - 处理结果
    
  server: # 工具提供者
    - 提供工具列表
    - 执行工具调用
    - 返回结果
    
  protocol: # 通信标准
    - JSON-RPC 2.0
    - 统一消息格式
    - 状态管理
```

### 4.2 工具定义

```json
// MCP工具定义示例
{
  "name": "read_file",
  "description": "读取文件内容",
  "inputSchema": {

    "properties": {
      "path": {
        "type": "string",
        "description": "文件路径"
      }
    },

  }
}
```

## 五、MCP实战：5分钟快速入门

### 5.1 安装MCP服务器

```bash
# 使用npm安装
npm install @anthropic-ai/mcp-server

# 或使用Docker
docker run -p 3000:3000 anthropic/mcp-server
```

### 5.2 配置MCP客户端

```typescript
import { Client } from '@anthropic-ai/mcp-client';

const client = new Client({
  servers: [
    {
      url: 'http://localhost:3000',
      auth: { type: 'bearer', token: 'your-token' }
    }
  ]
});
```

### 5.3 调用工具

```typescript
// 自动发现可用工具
const tools = await client.listTools();

// 调用工具
const result = await client.callTool('read_file', {
  path: './README.md'
});

console.log(result);
```
